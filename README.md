# Keymira

Keymira 是一款 Windows 平台的键盘监听与分析软件，提供实时按键显示、使用统计和热力图生成等功能。

## 功能特点

- 实时显示键盘按键
- 统计按键使用频率
- 生成键盘使用热力图
- 多用户支持
- 详细的使用统计报告
